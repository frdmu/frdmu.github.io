- close port
```
sudo fuser -k -n tcp 8080
```

- change gcc/g++ version
```
sudo update-alternatives --config gcc
sudo update-alternatives --config g++
```

- set up dual screen
```
xranr
// resolution
xrandr --output DVI-D-0 --mode 1280x1024
// set primary screen
xrandr --output HDMI-1-1 --primary
```
